Método de Clusterização Hierárquica com distância de Jaccard

Exemplo do lógica do cálculo da matriz de distâncias de Jaccard

# calcula matriz de distâncias de Jaccard ---------------------------------

for (j in 1:m) {
  for (k in 1:j) {
    if (j != k) {
      D[j, k] <- 1 - sum(X[, j] * X[, k]) / (sum(X[, j]) + sum(X[, k]) - sum(X[, j] * X[, k]))
      D[k, j] <- D[j, k]
    }
  }
}

MDS - Escalonamento Multidimensional

A técnica do escalonamento multidimensional foi usada para representar espacialmente em 2D uma matriz de proximidades entre uma série de objetos de modo que possamos visualizar mais facilmente os clusters de Academias

# gráfico em duas dimenssões para apresentar as dissimilaridades ---

mds <- cmdscale(D, eig = TRUE) 

hc <- hclust(dist(mds$points))

clusters <- cutree(hc, k = 7)

df <- data.frame(acad = rownames(mds$points),
                 coord1 = mds$points[,1],
                 coord2 = mds$points[,2],
                 cluster = factor(clusters))

df %>% 
  ggplot(aes(x = coord1, y = coord2, label = rownames(df))) +
  geom_text_repel(size = 4, force = 4) + 
  geom_point(aes(color = cluster), alpha = 0.55, size = 4, show.legend = FALSE) +
  labs(x = "Coordenada 1", y = "Coordenada 2") + 
  theme_bw()
